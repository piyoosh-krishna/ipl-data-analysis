# 🏏 IPL Data Analysis

### Exploring Team Performance, Match Trends, and Player Achievements

---

## 📌 Project Overview
This project analyzes IPL match data to uncover trends in team performance, toss impact, and player achievements.

---

## 📂 Dataset Used
**IPL Matches Dataset**
- Match details (teams, venue, season)
- Toss winner and decision
- Match winner
- Player of the match

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  

---

## 🔄 Project Workflow
1. Data loading and cleaning  
2. Data aggregation  
3. Exploratory Data Analysis (EDA)  
4. Visualization of trends  

---

## 📈 Key Insights
- Some teams consistently dominate  
- Toss decisions influence match outcomes  
- Certain players frequently win awards  

---

## 📊 Visualizations Included
- Top winning teams  
- Toss decision distribution  
- Matches per season  
- Player of match analysis  

---
